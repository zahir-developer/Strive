Main Menu

In the Menu Bar, the list of menu items is dynamically built when the user logs into the application. The list of pages that are included in the user’s profile determines the list of menu items that are displayed for the user.  If a user has not been granted access to pages under a menu, that menu will not appear when the user logs in.  The choices under a specific pull-down menu are also controlled by the list of pages included in a user’s profile
